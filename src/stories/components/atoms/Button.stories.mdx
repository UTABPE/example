# Кнопки

Кнопка – компонент, который побуждает пользователя произвести какое-то завершающее действие или сделать выбор. Например, перейти на следующую страницу или назад, дать согласие на обработку данных, подтвердить действие и т.п.

### Installation

```js
import { Button } from '@components/atoms/Button';
```

#### Основная кнопка (синяя)

Используется, когда необходимо обозначить ключевое или стартовое действие на странице. Например, начать заполнение формы или завершить работу с формой. На одной странице не может использоваться свыше одной основной кнопки. Также кнопки используются в диалоговом окне, когда интерфейс предлагает совершить те или иные альтернативные действия. Например, продолжить работу или вернуться к редактированию.

#### Второстепенная кнопка (с обводкой)

Используется, когда компонент не должен привлекать повышенного внимания или когда место на странице ограничено для размещения, а использование ссылки не подходит. Кнопка может помещаться на одной строке с другими компонентами и не предполагает привлечения обязательного внимания всех пользователей.

We can place an icon inside the Button in either the leading or the trailing position to enhance the visual context. It is recommended to use an octicon here.
